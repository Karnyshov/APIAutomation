# APIAutomation

Project for studying principles of API Automation.

Hints:
1. Generate JUnit style report: pytest --junitxml=<path><name.xml>
2. Install allure for generating a report. For example, via Homebrew (macOS).
3. Generate Allure report: pytest --alluredir=/tmp/ ./tests; allure serve /tmp/
4. Use run.sh file for running in CI/CD. Checked on TeamCity.
